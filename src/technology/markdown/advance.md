## 表格

Markdown 使用三个连字符创建每列的标题，并使用\|来分隔每列。

```md
| 测试| 有限|
| --- | ---|
|你好| 你好|
|我爱你|你爱我|
```

呈现输出如以下所示：

> | 测试  | 有限  |
> |-----|-----|
> | 你好  | 你好  |
> | 我爱你 | 你爱我 |

对齐方式：

```
| 测试 | 有限 |你好|
|:----|:----:|----:|
| 你好 | 你好 |你好|
|我爱你|你爱我|蜜雪冰城甜蜜蜜|
```
第一种表示左对齐，第二种表示居中对齐，第三种表示右对齐。

演示效果：

> | 测试 | 有限 |你好|
> |:----|:----:|----:|
> | 你好 | 你好 |你好|
> |我爱你|你爱我|蜜雪冰城甜蜜蜜|

## 代码块

可以通过三个 \``` 来创建代码块。其后可以添加一种语言，也可以不添加。


\```python<br>
print('hello world')<br>
\```



效果如下所示：

```python
print('hello world')
```

## 列表

对需要定义的列表在前方添加冒号加空格。

```md
第一列
: 我

第三人称
: it
: he
: she
```
## 上下标

通过 `^` 和 `~` 来对文档添加上标和下标。

```md
a^2^, b~2~
```
a^2^, b~2~

## 删除线

使用两个~~来对文档添加删除线。

`~~你好~~`

演示效果：
~~你好~~

## 任务列表

通过一个-加空格加上[]来创建一个复选框，如果需要选中则在[]中添加一个x。

```md
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```

演示效果：

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media

## 数学公式

数学公式主要是通过 Latex 语法[^1] 来进行支持，主要分为两种，第一种是行内公式，第二种是单行公式。

[^1]:Latex 语法是一种专业的排版工具，不建议专门学习，需要的时候搜索即可。

```md
$\frac{a}{b}$
$$\frac{a}{b}$$
```

$\frac{a}{b}$
$$\frac{a}{b}$$

具体可以查看 Latex 数学公式。
